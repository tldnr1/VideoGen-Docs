## 삼성 VXT(Visual eXperience Transformation) 조사 ([VXT Menu Docs](https://docs.samsungvx.com/eng/001_vxtcmsoverview_1.html#0))

## 개요
- 스마트 사이니지를 위한 클라우드 기반 솔루션
- 연동된 스크린에 제작한 미디어 영상을 실시간으로 송출 가능


## 기본적인 기능
- 콘텐츠를 ppt나 canva처럼 편집할 수 있는 기능 제공
- 위젯 배치 시, 레이어를 통해 개별 lock 등으로 편집 보조 기능 제공
- 다양한 분야(교육, 헬스케어, 아트 이미지 등)에 기본적인 템플릿 제공
- 각 이미지에 대해 개별적인 애니메이션(회전, 확대 등) 적용 가능
    - 위젯에서 회전, 확장, 웨이브 애니메이션 추가로 제공됨

## 콘텐츠 생성 관련
- 미리보기를 통해 생성된 영상 확인 기능
- 생성된 영상이 `멈추지 않고 반복재생` 되도록 반복 송출 가능
- 생성할 영상의 길이를 각 페이지 별로 개별 지정 가능 (페이지 편집에서 재생 시간 변경)
- Pixabay 이미지 업로드 가능
- 위젯 관련
    - 소셜 미디어(인스타 피드, 유튜브 영상) 연동하여 보여주기
    - HDMI, 링크 등에서 들어오는 외부입력 데이터 연동
    - RSS, QR, 카운트다운 제공
    - 날씨, 현재시간, 시계 등이 실시간 정보 반영되어 변경됨
- 텍스트 스타일 다양하게 템플릿으로 제공 (네온, 그라데이션 등 움직이는 텍스트 형태로 존재)
- 영상 길이 관련
    - 편집 창에서 지정 가능한 시간은 최소 5초, 최대 1시간으로 제한되는 것으로 확인됨
    - 각 vxt player에서 제공되는 형식이나 기능들의 제한이 서로 다름 ([참고 문서](https://docs.samsungvx.com/eng/003_basicfunctions_20.html#0))
    - 해상도의 경우 기본적으로 1920x1080 제공되는 것으로 보임 (스크린 별 차이 존재)

## Apps (콘텐츠 편집 페이지 중)
- MS OneDrive, Google Drive, Excel CSV 등 연동
- Vistar Media 연동
- 삼성 SmartThings, Data Sync 기능 제공
- Link My POS 연동 가능
- Ngine Real Estate, Ngine Real Automotive 연동 기능 제공 (특정 국가에서 제한됨; 영국, 스페인 등)

## AI 관련 기능 (챗봇 제외 Apps에서 설치하여 사용)
- 기본 홈페이지에 간단한 문의사항 답변해주는 챗봇 존재
- Text To Image로 `원하는 이미지(Image Generator)`, `ppt 템플릿 페이지(AI CorpPost)` 생성 기능 제공
- `ShadowGen` 을 통한 그림자 생성 기능 제공
- 저작권 해결된 음악, 배경 영상 제공해주는 서비스
- 생성 기능들에 대해 Aspect Ratio로 가로, 세로를 기본적으로 지원 (이미지의 경우 Realistic, Art, Sketch 스타일 제공)

## 사용 후기
1. 장점
    - 템플릿 위에 간단한 편집으로 접근 가능
    - 클라우드 서비스라 서버 구축 없이 사용 가능
    - 미리보기를 통해 결과물 확인 가능
    - 생성, 배포, 관리를 한 사이트에서 조작 가능하다는 장점
<br>

2. 단점
    - 편집한 내용이 실시간 송출이 된다는 것 외에 크게 부각되는 장점이 없음 (Canva와 비교했을 때 편집 면에서는 제공되는 기능이 적고 불편함)
    - 다양한 기능이 제공되는 것 같은데 각각에 대해 접근이 어려움 (찾아서 사용하기 힘들다고 생각함)
    - 챗봇을 제외한 AI 기능이 미흡함
        - 이미지 생성의 경우 단순한 입력으로는 퀄리티가 매우 낮음
        - 그림자 생성의 경우 올바른 위치에 생성 자체가 잘 되지 않음
        - 템플릿 생성이나 다른 기능에서 프롬프트 이해를 못해서 생성이 되지 않거나 생성 시간이 오래 걸리는 경우가 잦음
    - 페이지 전환이나 위젯을 옮기는 등 조작 시 반응이 느린 편
    - 기본적인 편집을 제외한 기능들이 Beta(AI 생성 기능) 단계이거나 결제를 하지 않으면 접근이 불가능함
        - 특히 `내보내기` 기능이 유료 플랜으로 묶여있어 활용에 제한됨
<br>

3. 개인적인 생각
    - 클라우드로 서비스가 제공되고 실시간 연동이 된다는 장점이 있긴 하지만, 실제 여러 매장에서 직접 연결하는 방식보다 비용이 높은 것 같음
    - 템플릿 기반의 편집 방식이라 세밀한 묘사나 원하는 디자인의 부재가 자주 발생할 것으로 보임
    - Gen AI 기능의 퀄리티가 낮아 이용 가능한 부분이 적음
    - 서비스 목적 상, 사용자의 의도에 맞는 추천 기능이 있으면 좋을 것 같음 (한줄 문구 추천, 템플릿 레이아웃 추천 등)
